# DECCAN-AI

# 🚀 AI-Powered Skill Assessment & Learning Agent

## 💡 Problem

Resumes only show claimed skills, not actual proficiency.

## 🎯 Solution

This system:

* Extracts skills from JD & Resume
* Conducts a conversational assessment
* Evaluates real proficiency
* Identifies skill gaps
* Generates personalized learning plans

## 🧠 Features

* 💬 Chat-based AI interview
* 📊 Skill scoring (0–10)
* 🎯 Gap detection
* 📚 Learning roadmap with time estimates
* 🌐 Works for multiple domains

## ⚙️ Tech Stack

* Python (Flask)
* HTML/CSS/JS
* Rule-based NLP logic

## 🏗️ Architecture

Input → Skill Extraction → Question Generator → Scoring Engine → Learning Plan Generator
